# Mysql-Employee-Database
MySQL - DDL Commands, Constraints

This project focuses on designing and managing an Employee Database using MySQL. The objective is to create database tables, establish relationships between tables, modify table structures using DDL commands, and apply constraints to maintain data integrity and consistency.

The database contains information about:

Departments
Locations
Employees

Primary keys and foreign keys are used to establish relationships between tables, while constraints ensure valid and reliable data entry.
